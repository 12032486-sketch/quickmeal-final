# QuickMeal Finder 🍽️

## 📌 Description
QuickMeal Finder is a React-based web application that generates random meal ideas instantly. It helps users quickly decide what to eat using a clean and modern interface.

## 🚀 Features
- Random meal generator
- Save favorite meals
- Responsive design
- Modern UI with glass effects
- Multi-page navigation

## 🛠️ Technologies Used
- ReactJS
- Vite
- CSS
- React Router

## ⚙️ Setup
1. Install dependencies:
   npm install

2. Run project:
   npm run dev
